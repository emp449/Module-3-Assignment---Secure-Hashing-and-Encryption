# Crypto App (Python)

This project demonstrates **basic cryptography** in Python:

1. **SHA-256 Hashing**  
   - Hash a string or a file using SHA-256.

2. **Caesar Cipher**  
   - Encrypt/decrypt text with a simple substitution cipher.

3. **Digital Signatures (RSA)**  
   - Generate RSA keypair
   - Sign text or files
   - Verify signatures
